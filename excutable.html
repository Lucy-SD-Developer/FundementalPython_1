Make your own .exe

  • Please refer to pyinstaller documentation: How do I make an executable from a Python script?
  • Another reference: see here
  1. Create a new folder/directory, rename it as 'firstExe'
  2. Prepare one python script file that has GUI, like your turtle program, easygui, tkinter...
  3. Prepare one xxxx.ico image that will be your .exe icon (refer to this website)
  4. Put the zzzz.py file and .ico image under the same folder: firstExe
  5. pip install pyinstaller
  6. If you are using windows OS, please install pywin32
  7. In your terminal/command prompt, input pyinstaller -h to inspect parameters
  8. Change current directory to 'firstExe'
  9. Input: pyinstaller -F -w --icon=xxxx.ico zzzz.py
  10. Press enter key
  11. Find your .exe in dist directory
  12. Remember the .exe can only work on the same OS
